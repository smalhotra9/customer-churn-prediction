# Customer Churn Prediction

## Overview
A machine learning project that predicts customer churn for a telecom company 
using a dataset of 7,000+ customer records.

## Problem Statement
Customer churn is costly for businesses. This project builds a classification 
model to identify customers likely to churn so the business can intervene early.

## Models Used
- Logistic Regression
- Random Forest
- Decision Tree
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- XGBoost

## Best Result
Logistic Regression achieved a recall of 0.82 for churners using a 
custom probability threshold of 0.4.

## Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn

## Project Structure
- Exploratory Data Analysis (EDA)
- Data preprocessing and encoding
- Train/Validation/Test split
- Model training and comparison
- Hyperparameter tuning with GridSearchCV
- Evaluation using classification reports, confusion matrices and ROC curves
